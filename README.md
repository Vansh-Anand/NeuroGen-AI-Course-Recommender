# NeuroGen

A robust system designed to elevate student success by predicting academic outcomes, recommending tailored electives, and proactively intercepting at-risk trajectories using interpretable AI.

🚀 **Repository:** [https://github.com/Vansh-Anand/NeuroGen-AI-Course-Recommender](https://github.com/Vansh-Anand/NeuroGen-AI-Course-Recommender)

A modernized educational intelligence platform built using React, Tailwind CSS, Recharts, Framer Motion, Python, Flask, and Scikit-learn.

## ✨ Features

### 📡 **PRWF Architecture Integration**
- **Predict**: Highly accurate academic outcome predictions (Distinction, Pass, Fail, Withdrawn) generated by an ensemble ML model (Neural Networks, Random Forest, Gradient Boosting).
- **Recommend**: Intelligent elective sorting leveraging historical course data, demographic clustering, and past performance.
- **Warn**: Real-time neuro-fuzzy logic system that quantifies academic risk levels and highlights struggling students.
- **Fair**: Fairness auditing metrics ensuring that recommendations do not demonstrate disparate impact across gender, region, or disability demographics.

### 🧠 **Neuro-Fuzzy Risk Assessment**
- **Hybrid AI Logic**: Blends Neural Network probability outputs with human-interpretable fuzzy logic rules.
- **Actionable Insights**: Converts raw probabilistic multi-class outputs into graded risk scores and actionable feedback labels (e.g., "Critical Risk", "On Track").

### 📊 **Interactive Analytics Dashboard**
- **Deep Explanations**: Interactive breakdown of feature contributions to show exactly *why* a student was flagged.
- **Dynamic Visualizations**: Utilizes Recharts and Framer Motion to map the student's neural probability network and prediction confidence smoothly.
- **Performance History Tracking**: Deep analysis tabs that weigh historical grades to refine future recommendations.

### 🎨 **Modern UI/UX**
- **High-Performance Interface**: Vite and React 18 power an exceptionally fast dashboard.
- **Responsive Animations**: GSAP and Framer Motion provide sleek micro-interactions and loading transitions.
- **Glassmorphism & Gradients**: Designed with a premium, engaging aesthetic that makes deep data exploration accessible.

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- Python 3.9+
- Git

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Vansh-Anand/NeuroGen-AI-Course-Recommender.git
   ```

2. **Backend Setup (Python)**
   Open a terminal in the root directory:
   ```bash
   # Create a virtual environment
   python -m venv .venv
   
   # Activate the environment
   # Windows:
   .venv\Scripts\activate
   # Mac/Linux:
   source .venv/bin/activate
   
   # Install dependencies
   pip install -r requirements.txt
   
   # Start the Flask API
   python api.py
   ```
   *Note: Due to size constraints, the `.pkl` ML model wrapper files must be generated or downloaded locally to execute `api.py` successfully.*

3. **Frontend Setup (Node)**
   Open a separate terminal and navigate to the frontend:
   ```bash
   cd frontend
   
   # Install dependencies
   npm install
   
   # Run the development server
   npm run dev
   ```

4. **Open your browser**
   Navigate to [http://localhost:5173](http://localhost:5173) to view the dashboard!

## 🌟 Key Components

### **prwf_architecture.py & neuro_fuzzy_system.py**
- Core engine files handling the data pipeline from raw demographic data into the SSV (Student State Vector) through neural probability mapping and fuzzy boundary checking.

### **api.py**
- Centralized Flask server utilizing `joblib` model loading designed to process POST requests with sub-second latency to feed the React frontend.

### **OutcomePrediction & Explainability (React Components)**
- The heart of the UI; receives the JSON predictions and maps them to user-friendly gauges, probabilistic bars, and feature-importance metrics highlighting engagement, scores, and demographic weight.
